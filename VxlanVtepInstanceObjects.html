VxlanVtepInstance Object¶

config/VxlanVtepInstance¶

  • Multiple objects of this type can exist in a system.
PARAMETER NAME DATA TYPE DESCRIPTION DEFAULT VALID VALUES
Vni [KEY] uint32 VXLAN Network ID N/A N/A
Intf [KEY] string VTEP instance identifier name. should be defined as either vtep<id#> or <id#> if the later then ‘vtep’ will be prepended to the <id#> example N/A N/A
DstIp string Destination IP address for the static VxLAN tunnel N/A N/A
IntfRef string Source interface where the source ip will be derived from. If an interface is not supplied the src-ip will be used. This attribute takes presedence over src-ip attribute. N/A N/A
Mtu uint32 Set the MTU to be applied to all VTEP within this VxLAN 1550 N/A
SrcIp string Source IP address for the VxLAN tunnel 0.0.0.0 N/A
TTL uint16 TTL of the Vxlan tunnel 64 N/A
VlanId uint16 Vlan Id to encapsulate with the vtep tunnel ethernet header N/A N/A
AdminState string Administrative state of VXLAN MAC/IP layer DOWN UP, DOWN
DstUDP uint16 vxlan udp port. Deafult is the iana default udp port 4789 N/A
InnerVlanHandlingMode int32 The inner vlan tag handling mode. 0 DISCARD_INNER_VLAN(0), NO_DISCARD_INNER_VLAN(1)
TOS uint16 Type of Service 0 N/A
