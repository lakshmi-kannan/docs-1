Fan Object
=============================================================

*config/Fan*
------------------------------------

- Multiple objects of this type can exist in a system.

+--------------------+---------------+----------------------+-------------+------------------+
| **PARAMETER NAME** | **DATA TYPE** |   **DESCRIPTION**    | **DEFAULT** | **VALID VALUES** |
+--------------------+---------------+----------------------+-------------+------------------+
| FanId **[KEY]**    | int32         | Fan unit id          |           0 | N/A              |
+--------------------+---------------+----------------------+-------------+------------------+
| AdminSpeed         | int32         | Fan set speed in rpm | N/A         | N/A              |
+--------------------+---------------+----------------------+-------------+------------------+
| AdminState         | string        | Fan admin ON/OFF     | N/A         | N/A              |
+--------------------+---------------+----------------------+-------------+------------------+



*FlexSwitch CURL API Supported*
------------------------------------

	- GET By Key
		 curl -X GET -H 'Content-Type: application/json' --header 'Accept: application/json' -d '{<Model Object as json-Data>}' http://device-management-IP:8080/public/v1/config/Fan
	- GET By ID
		 curl -X GET http://device-management-IP:8080/public/v1/config/Fan/<uuid>
	- GET ALL
		 curl -X GET http://device-management-IP:8080/public/v1/config/Fans?CurrentMarker=<x>&Count=<y>
	- UPDATE(PATCH) By Key
		 curl -X PATCH -H 'Content-Type: application/json' -d '{<Model Object as json data>}'  http://device-management-IP:8080/public/v1/config/Fan
	- UPDATE(PATCH) By ID
		 curl -X PATCH -H 'Content-Type: application/json' -d '{<Model Object as json data>}'  http://device-management-IP:8080/public/v1/config/Fan<uuid>
