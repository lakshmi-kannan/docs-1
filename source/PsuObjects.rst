Psu Object
=============================================================

*config/Psu*
------------------------------------

- Multiple objects of this type can exist in a system.

+--------------------+---------------+-------------------+-------------+------------------+
| **PARAMETER NAME** | **DATA TYPE** |  **DESCRIPTION**  | **DEFAULT** | **VALID VALUES** |
+--------------------+---------------+-------------------+-------------+------------------+
| PsuId **[KEY]**    | int32         | PSU id            |           0 | N/A              |
+--------------------+---------------+-------------------+-------------+------------------+
| AdminState         | string        | Admin UP/DOWN PSU | N/A         | N/A              |
+--------------------+---------------+-------------------+-------------+------------------+



*FlexSwitch CURL API Supported*
------------------------------------

	- GET By Key
		 curl -X GET -H 'Content-Type: application/json' --header 'Accept: application/json' -d '{<Model Object as json-Data>}' http://device-management-IP:8080/public/v1/config/Psu
	- GET By ID
		 curl -X GET http://device-management-IP:8080/public/v1/config/Psu/<uuid>
	- GET ALL
		 curl -X GET http://device-management-IP:8080/public/v1/config/Psus?CurrentMarker=<x>&Count=<y>
	- UPDATE(PATCH) By Key
		 curl -X PATCH -H 'Content-Type: application/json' -d '{<Model Object as json data>}'  http://device-management-IP:8080/public/v1/config/Psu
	- UPDATE(PATCH) By ID
		 curl -X PATCH -H 'Content-Type: application/json' -d '{<Model Object as json data>}'  http://device-management-IP:8080/public/v1/config/Psu<uuid>
