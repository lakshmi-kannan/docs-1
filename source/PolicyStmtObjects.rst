PolicyStmt Object
=============================================================

*config/PolicyStmt*
------------------------------------

- Multiple objects of this type can exist in a system.

+--------------------+---------------+--------------------------------+-------------+------------------+
| **PARAMETER NAME** | **DATA TYPE** |        **DESCRIPTION**         | **DEFAULT** | **VALID VALUES** |
+--------------------+---------------+--------------------------------+-------------+------------------+
| Name **[KEY]**     | string        | Policy Statement Name          | N/A         | N/A              |
+--------------------+---------------+--------------------------------+-------------+------------------+
| Action             | string        | Action for this policy         | deny        | permit, deny     |
|                    |               | statement                      |             |                  |
+--------------------+---------------+--------------------------------+-------------+------------------+
| Conditions         | string        | List of conditions added to    | N/A         | N/A              |
|                    |               | this policy statement          |             |                  |
+--------------------+---------------+--------------------------------+-------------+------------------+
| MatchConditions    | string        | Specifies whether to match     | all         | any, all         |
|                    |               | all/any of the conditions of   |             |                  |
|                    |               | this policy statement          |             |                  |
+--------------------+---------------+--------------------------------+-------------+------------------+

		 curl -X GET http://device-management-IP:8080/public/v1/config/PolicyStmt/<uuid>

	- DELETE By Key
		 curl -X DELETE -i -H 'Accept:application/json' -d '{<Model Object as json data>}' http://device-management-IP:8080/public/v1/config/PolicyStmt
	- DELETE By ID
		 curl -X DELETE http://device-management-IP:8080/public/v1/config/PolicyStmt<uuid>
	- UPDATE(PATCH) By Key
		 curl -X PATCH -H 'Content-Type: application/json' -d '{<Model Object as json data>}'  http://device-management-IP:8080/public/v1/config/PolicyStmt
	- UPDATE(PATCH) By ID
		 curl -X PATCH -H 'Content-Type: application/json' -d '{<Model Object as json data>}'  http://device-management-IP:8080/public/v1/config/PolicyStmt<uuid>
