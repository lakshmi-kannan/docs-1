CoppStatState Object¶

state/CoppStat¶

  • Multiple objects of this type can exist in a system.
PARAMETER NAME DATA TYPE DESCRIPTION DEFAULT VALID VALUES
Protocol [KEY] string Protocol type for which CoPP is configured. N/A N/A
RedPackets int64 Dropped packets. Packets marked with red for tri color policer. N/A N/A
BurstRate int32 Burst rate (packets) for policer. N/A N/A
GreenPackets int64 Packets marked with green for tri color policer. N/A N/A
PeakRate int32 Peak rate (packets) for policer. N/A N/A

FlexSwitch CURL API Supported¶

  • GET By Key

    curl -X GET -H ‘Content-Type: application/json’ –header ‘Accept: application/json’ -d ‘{<Model Object as json-Data>}’ http://device-management-IP:8080/public/v1/state/CoppStat

  • GET ALL

    curl -X GET http://device-management-IP:8080/public/v1/state/CoppStats?CurrentMarker=<x>&Count=<y>

  • GET By ID

    curl -X GET http://device-management-IP:8080/public/v1/config/CoppStatState/<uuid>
